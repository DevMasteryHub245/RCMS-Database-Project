# RCMS-Database-Project
A fully functional relational database for a multi-branch  restaurant chain. Built using Oracle 10g XE with 11 normalised  tables (3NF), complete DDL/DML scripts, 10 business queries  (JOINs, subqueries, aggregation), a View, Trigger, and Stored  Procedure. Final project for Introduction to Database Systems  course at IIUI Islamabad.
